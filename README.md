## Installation
To run this project locally:
1. Clone the repository:
   ```sh
   git clone https://github.com/samimalikdev/realtime-chat-app.git
   ```
2. Navigate to the project folder:
   ```sh
   cd realtime-chat-app
   ```
3. Install dependencies:
   ```sh
   flutter pub get
   ```
4. Run the app:
   ```sh
   flutter run
   ```
